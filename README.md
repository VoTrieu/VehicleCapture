# Vehicle Capture

Vehicle Capture is a beginner-friendly native Android app built with Kotlin and XML layouts. It lets users capture basic vehicle information, take a vehicle photo, save records locally, and manage saved vehicles.

This project was created as a Junior Android Developer portfolio project inspired by vehicle inspection capture workflows.

## Features

- View saved vehicles on the home screen
- Add a vehicle with make/model, year, mileage, and notes
- Take a vehicle photo using the device camera
- Preview the captured photo
- Save vehicle records locally
- Display saved vehicles in a RecyclerView
- Edit existing vehicle records
- Delete vehicle records with confirmation
- Basic input validation
- Custom Material toolbar and status bar styling

## Tech Stack

- Kotlin
- XML layouts
- ViewBinding
- Data Binding

- Material Components
- Camera Intent
- FileProvider
- SharedPreferences
- Gson
